# VÅR STYRMODELL

**Styrmodellen är Smålands Fotbollförbunds gemensamma karta för styrning, ansvar och arbetssätt. Den hjälper distriktet att hitta rätt nivå, rätt mandat och rätt väg genom organisationen.**

Den ska ge ordning utan att göra arbetet krångligare än det behöver vara: stadig nog att hålla över tid och tydlig nog att användas i vardagen.

{% hint style="info" %}
**Introduktionsfilm kommer här**  
Här lägger vi in filmen där styrmodellen presenteras. Filmen bör kort berätta varför modellen finns, hur den ska användas och hur den hänger ihop med dokumentarkivet.
{% endhint %}

## För vem styrmodellen finns

Styrmodellen är i första hand ett internt stöd för personer och organ som har ansvar i distriktets styrning och verksamhet. Den publiceras öppet för att föreningar och andra intressenter ska kunna se hur distriktet arbetar.
