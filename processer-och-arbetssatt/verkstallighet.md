# Verkställighet

Verkställighet är när beslut blir arbete.

Ett beslut är inte färdigt bara för att det är fattat. Det behöver genomföras inom de ramar, resurser och tidsgränser som beslutet anger. Verkställigheten ska göra beslutet verkligt, inte skapa nya beslut i skymundan.

{% hint style="info" %}
Verkställighet betyder att genomföra ett beslut som redan är fattat. Om genomförandet kräver ett nytt självständigt vägval ska frågan tillbaka till rätt beslutsnivå. Hur sådana beslut fattas beskrivs i [beslutsfattande](beslutsfattande.md).
{% endhint %}

## Det beslutade sätter ramen

Den som verkställer ska hålla sig inom beslutets syfte och ramar. Det handlar framför allt om vad beslutet skulle uppnå, vilka ekonomiska ramar som gäller, när arbetet ska vara genomfört och hur återrapportering ska ske.

När ramen är tydlig blir verkställigheten både friare och tryggare.

## Vem som genomför beror på beslutet

Ofta är det [distriktschef](../distriktsstyrelsen-och-ledningen/distriktschef.md) och [kansli](../distriktsstyrelsen-och-ledningen/kansli.md) som verkställer beslut från distriktsstyrelsen och andra organ. Kommittéer kan bidra inom sina sakområden när uppdraget eller delegationen säger det.

Den konkreta ansvarsfördelningen beskrivs i [ansvarsmatrisen](../strategi-och-operativt-uppdrag/ansvarsmatris.md) och i berörda dokument i [dokumentarkivet](https://styrning.smaland.football/dokument).

## När ramen inte räcker

Ibland visar det sig under arbetets gång att beslutet inte räcker. Då ska frågan inte lösas informellt.

{% stepper %}
{% step %}
**Ramen håller**  
Genomför arbetet enligt beslut, budget, delegation och tidsplan.
{% endstep %}

{% step %}
**Ramen är oklar**  
Be om förtydligande från rätt nivå innan arbetet går vidare.
{% endstep %}

{% step %}
**Ramen behöver ändras**  
Lyft frågan tillbaka till det organ eller den funktion som har mandat att fatta nytt beslut.
{% endstep %}

{% step %}
**Avvikelsen behöver hanteras**  
Återrapportera enligt principerna i [uppföljning och återrapportering](uppfoljning-och-aterrapportering.md).
{% endstep %}
{% endstepper %}

## Verkställighet ska kunna följas

Den nivå som har gett uppdraget eller fattat beslutet behöver kunna följa vad som händer efter beslutet. Mer om uppföljning, återrapportering och avvikelsehantering finns i [uppföljning och återrapportering](uppfoljning-och-aterrapportering.md).

Verkställighet fungerar när den som genomför vet vad som gäller, den som beslutat får återrapportering och organisationen kan se om arbetet leder dit det var tänkt.
