# Beslutsfattande

Beslut ska fattas där mandatet finns.

Det är en enkel princip, men den behöver hållas levande. Ett beslut blir inte bättre av att fattas snabbt om det fattas på fel nivå.

{% hint style="info" %}
Om det är oklart vem som får besluta ska frågan prövas mot [normordningen](../grund-och-varden/normordning.md), [delegationen](../distriktsstyrelsen-och-ledningen/delegation.md) och [ansvarsmatrisen](../strategi-och-operativt-uppdrag/ansvarsmatris.md).
{% endhint %}

## Ett beslut behöver bära sin egen tyngd

För att ett beslut ska vara legitimt behöver det vila på rätt nivå, rätt mandat och tillräcklig [beredning](beredning.md).

<table data-view="cards"><thead><tr><th></th></tr></thead><tbody><tr><td><strong>Rätt nivå</strong><br>Frågan hör hemma hos det organ eller den funktion som beslutar.</td></tr><tr><td><strong>Rätt mandat</strong><br>Beslutet har stöd i stadgar, demokratiskt beslut, delegation eller annat gällande dokument.</td></tr><tr><td><strong>Rätt underlag</strong><br>Ärendet är berett i den omfattning frågan kräver.</td></tr><tr><td><strong>Rätt nästa steg</strong><br>Det är tydligt om beslutet ska verkställas, återrapporteras eller följas upp.</td></tr></tbody></table>

## Beslut kan se olika ut

Alla beslut är inte av samma slag. Därför behöver organisationen skilja på vilken typ av beslut det handlar om.

{% columns %}
{% column %}
### Medlemsbeslut

Fattas av [årsmöte](../demokratisk-styrning/arsmote.md) eller [representantskap](../demokratisk-styrning/representantskap.md) inom respektive mandat.
{% endcolumn %}

{% column %}
### Styrelsebeslut

Fattas av [distriktsstyrelsen](../distriktsstyrelsen-och-ledningen/distriktsstyrelse.md) inom dess strategiska och organisatoriska ansvar.
{% endcolumn %}
{% endcolumns %}

{% columns %}
{% column %}
### Delegerade beslut

Fattas av organ, funktion eller person som fått tydlig [delegation](../distriktsstyrelsen-och-ledningen/delegation.md).
{% endcolumn %}

{% column %}
### Operativa beslut

Fattas inom löpande verksamhet och [verkställighet](verkstallighet.md), inom beslutade ramar.
{% endcolumn %}
{% endcolumns %}

## Verkställighet får inte bli nya beslut i skymundan

Det är viktigt att skilja mellan beslut och [verkställighet](verkstallighet.md).

Verkställighet innebär att genomföra något som redan är beslutat. Om genomförandet kräver ett nytt vägval, ändrad inriktning, ny kostnad, ny risk eller förändrat mandat kan det vara ett nytt beslut.

Då ska frågan tillbaka till rätt nivå.

## Brådska ändrar inte mandatet

Brådskande beslut kan behöva fattas mellan ordinarie sammanträden. Det ska vara undantag, inte normalt arbetssätt.

{% hint style="warning" %}
Brådska får inte användas för att kringgå rätt beslutsnivå, beredning, jävshantering eller dokumentation.
{% endhint %}

Beslut som fattas genom skriftligt förfarande, digital omröstning eller annan motsvarande ordning ska dokumenteras enligt [dokumentation och protokoll](dokumentation-och-protokoll.md).

## Ett beslut ska gå att använda

Ett beslut behöver vara så tydligt att nästa steg kan genomföras. Dokumentationen av beslutet hör hemma i [dokumentation och protokoll](dokumentation-och-protokoll.md), medan återrapportering och uppföljning beskrivs i [uppföljning och återrapportering](uppfoljning-och-aterrapportering.md).

När beslut fattas på rätt nivå och går att använda i vardagen blir styrningen mindre personberoende och mer hållbar.
